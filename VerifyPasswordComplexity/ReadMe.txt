VerifyComplexity Instructions

1. Double-Click VerifyComplexity.bat file
2. Browse to the VerifyComplexity.ps1 file
3. Select the User file
4. Click Open
5. Browse to the folder where the reports will be saved, click Open
6. in the PowerShell Window, enter the name of the file for the report
7. Press the Enter key on the keyboard
8. Wait for the file to be created
9. Wait for the PowerShell Window to close
10. In the cmd window, press any key to exit the Pause
